「学习笔记」使用hugo搭建个人静态博客

2021年4月26日 发布在 Blog, 学习笔记

-首先声明，本人小白，文章纯属个人学习笔记。内容学习自B站CodeSheep老师的视频。

-操作系统为MacOS

一、为什么搭建个人静态博客网站

  1. 可根据个人喜好对网页进行布局和规划，文章分类具体化，界面更美观，使用更自由。
  2. 动态网站功能强大但较为复杂，要求一定的前后端知识基础；需要个人服务器和域名，维护成本高。
  3. 静态网站搭建要求简单，远端服务器可连接github或者gitee。根据教程一步一步操作即可，适合我这样的小白。

二、软件准备

1. 使用终端安装Homebrew

Homebrew是一款自由及开放源代码的软件包管理系统，用以简化macOS系统上的软件安装过程。因其可扩展性得到了一致好评，而在Ruby on Rails社区广为人知。

安装指南可参考乔一亖—mac下高效安装 homebrew 及完美避坑姿势

最终本人采用官方源进行安装：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

中途出现两次报错：

错误1-curl: (7) Failed to connect to raw.githubusercontent.com port 443: Connection refused

解决方法一：

  • 系统偏好设置-网路-高级-DNS-添加DNS服务器8.8.8.8或114.114.114.114

解决方法二：

  • 打开IP查询网站: https://www.ipaddress.com/ 查询raw.githubusercontent.com的IP地址

  • 在终端修改hosts

sudo vim /etc/hosts

错误2-curl: (35) LibreSSL SSL_connect: SSL_ERROR_SYSCALL in connection to raw.githubusercontent.com:443

查了半天不知道怎么弄，很多人说是代理出了问题。但是我在终端输入vi ~/.gitconfig查看，并没有[http] 或者 [https]的相关配置。
最后参考：https://blog.csdn.net/wufantastic/article/details/91488651 解决

2. 下载安装Git（官方安装指南）

  • MacOS系统在终端里首次运行 git 命令即可跳出安装提示

3. 下载安装hugo（Hugo中文文档）

Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。

  • Mac下直接在终端使用 Homebrew 安装：
brew install hugo


-输入 hugo version
-返回hugo对应的版本，确认安装完成

三、使用hugo生成博客

1. 在终端输入语句建立站点：

hugo new site myblog

-文件夹名“myblog”可自定义。
-成功后会返回文件所在目录：

2. 下载并设置主题

主题的设置较为关键，界面和布局的选择全凭个人喜好。功能多的主题配置起来可能会复杂一些，说明文档也有所不同。
-hugo官网有众多精美的主题可供选择：https://themes.gohugo.io

-挑选一个自己喜欢的主题后，可以在 GitHub 仓库的 Release 页面找到最新的稳定版。 下载主题后解压，并放到站点目录的 theme 文件夹下。
也可以使用Git 命令来下载主题的最新版本，在建立的Hugo站点的theme目录下执行(有些主题要求在根目录下执行，具体看对应的文档说明)：

3. 使用主题Demo的内容

-一般下载主题都会带有exampleSite这个文件夹，需要把里面的所有内容复制，然后覆盖到根目录中

4. 在本地启动个人博客

-回到根目录，使用 hugo server 命令来启动博客

-返回一个本地链接，可以在浏览器打开，代表博客已经成功在本地启动！
